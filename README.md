## Firebase Configuração

1. No Firebase Console, adicione seu app Android e iOS.
2. Baixe `google-services.json` e coloque em `android/app/`.
3. Baixe `GoogleService-Info.plist` e coloque em `ios/Runner/`.
4. Ative o método de login por E-mail/Senha.
5. Execute:
   - `flutter pub get`
   - `flutter clean`
   - `flutter run`
